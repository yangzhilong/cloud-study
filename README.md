# 我的SpingCloud学习笔记
## 版本
```
	SpringBoot:	2.1.5.RELEASE
	SpringCloud: Greenwich.SR1
	SpringCloud-Alibaba: 2.1.0.RELEASE
```

## 项目模块：
### 1、cloud-parent：
```
 1、SpringBoot和SpringCloud的版本
 2、Plugin的版本
 3、Maven私服信息
 4、一些jar的version属性定义
```

### 2、common
```
 API项目和Service项目的公共类和Util合集
```

### 3、eureka-server
```
 eureka的server服务，有单机配置和集群配置
```

### 4、gateway
```
Gateway的各种集成
```

### 5、producer-service
```
服务的提供者的多个版本
```

### 5、consumer-service
```
服务的消费者的多个版本
```
